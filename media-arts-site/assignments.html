Assignments

A collection of Media Arts exercises exploring composition, editing, and visual storytelling.

Assignment 01: Visual Contrast Study

Exploration of light and shadow balance to create emotional tension in a single frame.

Photoshop tools used: Curves, Levels, Layer Mask

Assignment 02: Surreal Compositing

Layered photo elements to build a dream-like narrative around urban isolation.

Photoshop tools used: Pen Tool, Blending Modes, Gaussian Blur

Assignment 03: Motion Poster Draft

Poster concept focusing on directional flow, typography rhythm, and layered depth.

Photoshop tools used: Smart Objects, Gradient Map, Transform Warp

Assignment 04: Color Narrative Sequence

Four-frame sequence using color progression to communicate time and mood shifts.

Photoshop tools used: Selective Color, Hue/Saturation, Clone Stamp
